Firmware für M86:

Mit dem Objekt "Firmware" wird eine Firmware Datei zum aktuellen Package hinzugefügt.

Properties:
--description		...Beschreibung der Firmware Datei (wird nicht verwendet)
--outputFile_name	...Name folgender generierter Dateien: "mch", "ver"
--inputFile_directory	...Dateipfad zum Hexfile der Firmware (wichtig: Hexfile!)
--ver_major		...Major Version der Firmware
--ver_minor		...Minor Version der Firmware
--ver_subminor		...Subminor Version der Firmware
--verstate		...Entwicklungsstand

Beispiel:

description		= new Firmware
outputFile_name		= SAI205_FW
inputFile_directory	= C:\Documents\PackGen\A_BIOS_SAI205.h86
ver_major		= 01
ver_minor		= 05
ver_subminor		= 01
verstate		= Alpha
